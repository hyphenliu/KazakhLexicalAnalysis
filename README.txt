####################################################
#                                                              #
#                   【软件使用说明】                       #
#                                                              #
####################################################

1. 双击运行“安装必要软件.BAT”

2. 弹出软件安装向导，软件安装路径为系统默认的路径即可，
   如果更改路径很可能会导致本系统的其他软件不可使用，请
   谨慎选择！若要修改路径，请注意配置系统环境变量。

3. 安装相关软件后双击运行“创建快捷方式.py”，创建快捷方式。

4. 打开快捷方式运行程序。


【注意】
若出现问题，请自己动手安装packages文件夹内的软件，配置
系统环境变量，网上搜索怎么安装python第三方包。
